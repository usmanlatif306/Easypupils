# Easy Pupils

Easypupils is full school and college based application.

## Main Features

The best script for building the modern web fintech application.

- Super Admin
- Admin
- Student
- Admission
- Teacher
- Teacher Permission
- Parent
- Accountant
- Librarian
- Events
- Gallery
- Attendance
- Biomatric Attendance
- Daily Schedule
- Curriculum
- Slybus
- Class
- Class Rooms
- Departments
- Calender
- Marks
- Exams
- Grades
- Send Exam Marks to Parent and Student
- Expense Records
- Library Management
- Noticeboard
- Online Classes
- Chat With Teacher and Administration
- And Much More
